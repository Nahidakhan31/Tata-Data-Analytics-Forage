# Tata Data Analytics Job Simulation (Forage)

## Overview
This repository contains my work from the Tata iQ Data Analytics Job Simulation on Forage. 
The project focuses on analyzing delinquency risk, designing a predictive model, and proposing a responsible AI-powered collections system.

## Tasks Covered
- Exploratory Data Analysis (EDA) and data quality assessment
- Identification of key delinquency risk indicators
- Predictive modeling approach (conceptual)
- Business recommendations using SMART framework
- AI-powered collections system design
- Responsible and ethical AI considerations
